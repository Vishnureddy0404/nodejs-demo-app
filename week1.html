1. Introduction to Linux

Linux is an open-source operating system widely used in DevOps. It powers servers, cloud infrastructure, and development environments.

Common distributions include:

  • Ubuntu
  • CentOS
  • Red Hat Enterprise Linux (RHEL)
  • Alpine

2. Essential Linux Commands

These commands are fundamental for navigating and managing files in Linux.

# List files
ls -l

# Create a file
touch myfile.txt

# Move a file
mv myfile.txt /tmp/

# Remove a file
rm /tmp/myfile.txt

# View running processes
ps aux

3. File Permissions & User Management

File permissions determine who can read, write, or execute a file.

# Change permissions
chmod 755 script.sh

# Change ownership
chown user:user file.txt

# Add a new user
sudo useradd devops_user

4. Bash Scripting

Bash scripts automate tasks using loops, conditions, and functions.

#!/bin/bash
for i in {1..5}; do
    echo "Iteration $i"
done

5. Cron Jobs (Task Automation)

Schedule tasks to run at specific times using cron jobs.

# Open crontab
crontab -e

# Schedule a script to run every day at midnight
0 0 * * * /path/to/script.sh
